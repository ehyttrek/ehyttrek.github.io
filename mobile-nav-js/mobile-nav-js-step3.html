Overview

Media queries for basic responsive behaviour

In step 3 we are adding the media queries (or more precisely just one media query at the breakpoint of 600px) for the basic responsive behaviour: when below 600px viewport width the desktop menu is set to 'display: none' (one could say: 'invisible', even through it is really not there at all) and the mobile menu icon (the "burger icon") is set to 'display: block', i.e. 'visible'. Above 600px vice versa.

At this point there is no JavaScript yet to show and hide the actual mobile navigation. It is just media queries to toggle the two basic navigation elements.
