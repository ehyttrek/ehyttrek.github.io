Overview

All into functions

JavaScript functions is really where the power of that language lies. So in this final step, we control the entire behaviour of the mobile menu with two functions that are written into the script section in the head, and called in the actual navigation elements we click on.

Functions can be made event more lean and streamlined than this — with a little bit of knowledge and understanding of actual programming logic we could implement more variables, as well as arrrays and god knows what.

But it also works quite fine to fake your way through it with slightly more clumsy code …
