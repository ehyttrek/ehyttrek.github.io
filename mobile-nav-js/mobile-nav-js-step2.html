Overview

Preparing the two alternative menus with CSS

In this step, the main point is to add CSS styles to prepare the two alternative menus: one for desktop, one for mobile (please note, that the third menu now sits at the bottom, out of the way. It is really just for navigating the steps).

If you look at the CSS styles in the head of this document, you will note at the very top a selector to hide the desktop menu while we haven't yet set up the media queries for responsive behaviour. Feel free to make the desktop menu visible as described in the comment in the CSS. Of course you can only do that, if you copy the code to your own local versions of teh files — or wait until I give you the files.

Also note, that some CSS was added to make the entire page look a wee bit nicer. Some of this CSS is in an external style sheet — mainly the bits that are irrelevant for this exercise.
