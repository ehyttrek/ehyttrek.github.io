HTML Syntax

Tags

Tags are the essential building blocks of HTML. A standard HTML Tag consists of an opening tag and a closing tag following below syntax. 

    
<opening tag>content</closing tag>

Attributes

Attributes are important elements of further functionality of a tag. Attributes are always set in the opening tag. 

    
<div class="my-div">content</div>
