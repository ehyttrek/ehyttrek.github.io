What is this page about?

This page is my vault for examples and bits of HTML/CSS, which I use in classes and trainings I give on the topic. I mainly teach these topics on an introduction or basic level, so if you are a seasoned developer, please do not be surprised about the sometimes very basic level of the examples shown here.
