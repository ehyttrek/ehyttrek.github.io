'Old School': float

In the old days, before CSS columns, flex and grid (but after the HTML tables and pixel-GIF scaling), horzontal layouts would be done by abusing the float property.

This property was originally designed to make text float around an image, but soon it was the standard solution to make those darn block elements behave differently.

By declaring 'float: left' for the elements that were supposed to appear horizontally, you could make them do exactly that.

'True Columns': columns

With the column property you can actually create 'true columns', meaning: a continous block of content will flow into the specified amount of columns.

There are a number of ways to get the behaviour of the content in these columns under control, but some things are not easy to do.

'Stay Flexible': flex

Setting display to the value 'flex' is another way to create horizontal layouts

However, it doesn't let the content automatically flow into columns.

Flex in the default setting behaves very much like good old 'flow: left' -- except that you don't need a clearfix.
